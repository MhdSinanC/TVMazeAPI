# **TV Show Search App**

A simple web application that allows users to search for TV shows and view details such as titles, images, and ratings. The app fetches TV show data from the [TVMaze API](https://www.tvmaze.com/api), making it an easy way for users to explore popular shows.

## **Overview**

The **TV Show Search App** is a front-end project that provides an interface for users to search for TV shows by name. It displays relevant show information such as the show’s title, image, and average rating. If no shows match the search query, the app shows a "No Shows Found" message. This app is designed with a simple, clean layout to enhance the user experience.

---

## **Features**

* **Search TV Shows**: Search for TV shows by name using an easy-to-use search bar.
* **Show Details**: For each show, view the title, image, and average rating.
* **Responsive Design**: The layout is responsive and adjusts to different screen sizes.
* **Friendly UI**: Displays a "No Shows Found" message if no results are found for the search query.

---

## **Tech Stack**

This project uses the following technologies:

* **HTML** – Structure of the webpage.
* **CSS** – Styling and layout design.
* **JavaScript** – Handles the app logic, user interactions, and API requests.
* **Axios** – To make HTTP requests to the TVMaze API.
* **Font Awesome** – For displaying star icons in the ratings section.

## **Setup Instructions**

To run the project locally, follow these steps:

### 1. **Clone the Repository**

Clone the repository to your local machine:

```bash
git clone https://github.com/yourusername/tv-show-search-app.git
```

### 2. **Navigate to the Project Directory**

Go to the project folder:

```bash
cd tv-show-search-app
```

### 3. **Open `index.html`**

Open the `index.html` file in any browser. The app will work immediately without any additional setup.

## **Usage**

### 1. **Searching for TV Shows**

* In the search bar, type the title of the TV show you're looking for and hit **Search**.
* The app will display matching TV shows, including their titles, images, and ratings.
* If no shows match your query, a message will appear: **"No Shows Found :("**.

### 2. **Hover Effects**

* When you hover over a show card, the design will slightly change, offering a better user experience.
